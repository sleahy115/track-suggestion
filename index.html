Which programming language should I choose?

There are many different programming langauages. They each have thier own purpose's and a suited to different tasks. The first questions a prospective programmer usually asks is which language is best? Well there is no "best" language but there is a language that will be "best" for you. Answer the following questions for a suggestion on which language matches you.

Would you describe yourself as more artistic or more logical?

Would you perfer to work for a small start up, or for an established large company?

Would you perfer to crate wepages that are very interavtive for the user, or that have more fixed content>

Would you perfer to spend your time making something look great, or creating a framwork for it?

Would you like to learn how to create mobile apps?
