Which programming language should I choose?

There are many different programming languages. They each have their own purposes and a suited to different tasks. The first questions a prospective programmer usually asks is which language is best? Well, there is no "best" language but there is a language that will be "best" for you. Answer the following questions for a suggestion on which language matches you.

1.Would you describe yourself as more artistic or more logical?

2.Would you perfer to work for a small start up, or for an established large company?

3.Would you perfer to create wepages that are very interactive for the user, or that have more fixed content

4.Would you perfer to spend your time making something look great, or creating a framwork for it?

5.Would you like to learn how to create mobile apps?

You should study Java, you are interested in mobile development and you would like to work for a large company. Java is used for android development and is popular in large enterprises.
You should study PHP. You prefer a more fixed framework design on web pages and would prefer to work for a large company. PHP is popular in large enterprises including Facebook.
You should study Ruby. You would rather work for a small startup, Ruby is favored by smaller companies. You would also rather create something interactive, Ruby is the preferred language for interactive sites
You should study CSS/Design. CSS along with HTML5 are used to style websites. You are interested in being creative and artistic and the many many styling options in CSS will let you use those skills.
